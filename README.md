# House Price Prediction

##  Project Overview
This project predicts house prices based on features like location, square footage, number of bedrooms (BHK), and bathrooms.

---

##  Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- XGBoost

---

##  Workflow
1. Data Cleaning (handling missing values, removing outliers)
2. Feature Engineering (price per sqft, location grouping)
3. Encoding categorical variables (One-Hot Encoding)
4. Train-Test Split
5. Model Training (Linear Regression, Random Forest, XGBoost)
6. Model Evaluation (R² Score, RMSE)

---

##  Models Used
- Linear Regression (baseline)
- Lasso & Ridge
- Random Forest
- XGBoost

---

##  Results
- Ensemble models performed better than linear models
- Achieved good accuracy using feature engineering

---

##  Future Improvements
- Hyperparameter tuning
- Better feature selection
- Deploy as web app

---

## Dataset
Dataset taken from Kaggle (Bangalore House Data)
